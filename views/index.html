Ain't Nobody Got Time For That

Hit me up with the time, with a Unix timestamp or a natural language date (example: September 4, 1982)

If it does not contain a date or Unix timestamp, it returns null for those properties.

Example Usage:

https://alive-check.glitch.me/April%2019%202013

https://alive-check.glitch.me/1366329600

Example Output:

{"unix":"1366329600","natural":"April 19th 2013"}
